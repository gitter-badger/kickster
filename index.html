Easy installation

Install the Gem first: gem install kickster Made for your quick prototyping needs. Kickster has an easy and unobtrusive setup which gets you started in no time. Kickster uses Jekyll so ships with endless configuration possibilities.

{% highlight shell %} ~ $ gem install kickster ~ $ kickster new site_name ~ $ cd site_name ~ $ bin/setup {% endhighlight %}
